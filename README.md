## Sobre o Projeto

Este é um projeto simples e divertido que utiliza HTML, CSS e JavaScript para criar uma experiência interativa. O objetivo é permitir que o usuário descubra se é um Jedi ou um Sith com base no nome digitado.

##  Funcionalidades

* Input Interativo: Um campo onde o usuário pode digitar seu nome.
* Animação de Sabre de Luz: Um elemento div que se transforma em um sabre de luz azul (Jedi) ou vermelho (Sith) com efeito de brilho.
* Background Dinâmico: O plano de fundo do site muda para uma imagem temática de Jedi ou Sith, dependendo da escolha.
* Validação Simples: Alerta o usuário se um nome inválido for digitado.

## Tecnologias usadas

* HTML5: Estrutura da página.
* CSS3: Estilização, animações de transição e definição das cores e brilhos dos sabres.
* JavaScript: Lógica condicional para detectar o input do usuário e aplicar as classes CSS dinamicamente.
  
 ## Como Executar
Para visualizar o projeto, siga este passo simples:

  Clique no site na parte sobre do projeto
  ou


  1. Clone o repositório para sua máquina local.
  2. Abra o arquivo index.html no seu navegador.
  3. Digite Alexandre ou Proa no campo de input.
  4. Clique no botão "Entrar" para ver a Força agir!
